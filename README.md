# Motion Lab

A mobile-first web app for learning animation vocabulary through interactive demos.

## Run locally

Open `index.html` directly in a browser, or serve the folder:

```sh
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Features

- Searchable animation glossary grouped by category.
- Interactive stage for entrances, transforms, transitions, spring motion, looping motion, polish effects, and performance concepts.
- Controls for duration, delay, stagger, perspective, easing, and reduced motion.
