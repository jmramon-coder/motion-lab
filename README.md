# Motion Lab

A mobile-first web app for learning animation vocabulary through interactive demos, contextual controls, shareable links, and AI-friendly motion prompts.

## Run locally

Open `index.html` directly in a browser, or serve the folder:

```sh
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Features

- Animation library grouped by practical motion categories.
- Interactive stage for entrances, transforms, transitions, spring motion, looping motion, polish effects, and motion principles.
- Controls for duration, delay, stagger, perspective, easing, and reduced motion.
- Contextual controls for direction, distance, bounce, origin, item count, ticker target, and typewriter text.
- Copyable prompt, shareable state URL, and X-ready post copy for each animation.
- Subject gallery for testing motion against different visual assets.

## Launch angle

Motion Lab helps designers and developers describe motion more precisely, especially when prompting AI tools:

> Pick a term, see it move, tune it, and copy a better animation prompt.
